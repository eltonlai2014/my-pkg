# my-pkg

`my-pkg` 是一個 Go 函式庫，提供簡單的數學運算功能。

## 使用方式

```go
import "github.com/username/my-pkg/pkg/mylib"

result := mylib.Add(2, 3)
fmt.Println(result) // 5
